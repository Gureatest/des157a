journal post 6

date: november 8th, 2023

A point of interest that I found was how the article focuses primarily on mobile games on touchscreen devices. Mobile games are an interesting medium, as they take UI and interactivity much differently than a computer game or other mediums, such as console games. The way the user interacts is so indicative of what UI/UX rules to follow, whether a mouse and keyboard, a controller, or with touch.

Things like the “rate us” screen, or bigger icons all show how mobile design methods are so different than something like a computer game. However, there are some universal aspects of games that can be seen, such as hitbox sizes and visible buttons for optional interactivity. The cartoony style of mobile game UI is so interesting compared to the recent trend of minimalist UI of console and computer games.
